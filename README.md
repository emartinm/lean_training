# lean_training

Proyecto de juguete para aprender y practicar con Lean. Define un lenguaje imperativo pequeño (expresiones, booleanos y sentencias) en `Program.lean`, modela estados de ejecución y desarrolla su evaluación tanto en semántica de paso grande como de paso pequeño (`Semantics.lean`), junto con teoremas que relacionan ambos enfoques para practicar definiciones formales y demostraciones (`Semantics_th.lean`). 

También incluye algunas definiciones y teoremas usando listas y número naturales en `Basic.lean`
